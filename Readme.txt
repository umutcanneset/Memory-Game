The game is planned single or two-player.
There is main page which person can choose the mode.
Single player plays alone, on every match, it has a sound to specify it. 
For multiplayer,there are player one and player two points,
they are playing turn by turn. With every match, they have a distinct matching sound.
In any time, player can quit the game and return to main menu.

In the end of the game, there is a winning sound and there comes a button, for main menu.
